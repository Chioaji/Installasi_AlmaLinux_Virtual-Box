Jawaban nomor 3 = 
ext4 (Fourth Extended File System):
ext4 adalah sistem file yang umum digunakan pada distribusi Linux modern. Ini adalah penerus dari ext3 dan menyediakan berbagai perbaikan dan peningkatan kinerja.
ext3 (Third Extended File System):

ext3 adalah versi sebelumnya dari ext4 dan juga digunakan pada sistem operasi Linux. ext3 mendukung jurnal (journaling), yang membantu memulihkan sistem file setelah kegagalan sistem atau pemadaman yang tidak terduga.
swap:
Swap adalah jenis partisi yang digunakan oleh sistem operasi untuk menyimpan data yang sementara dipindahkan dari RAM ke penyimpanan sekunder, seperti hard drive. Ini membantu meningkatkan kinerja sistem dengan memberikan lebih banyak ruang untuk aplikasi dan proses.

ntfs (New Technology File System):
ntfs adalah sistem file yang banyak digunakan pada sistem operasi Windows, termasuk versi-versi lebih baru seperti Windows 10. Sistem operasi Linux juga dapat mendukung pembacaan dan penulisan pada partisi ntfs.

fat32 (File Allocation Table 32-bit):
fat32 adalah sistem file yang umum digunakan pada perangkat penyimpanan portabel, seperti USB flash drive dan kartu SD. Ini memiliki batasan ukuran file dan partisi tertentu, dan lebih umum digunakan pada perangkat yang memerlukan kompatibilitas lintas platform.

btrfs (B-tree File System):
btrfs adalah sistem file yang dikembangkan untuk Linux dan dirancang untuk menyediakan fitur-fitur canggih seperti manajemen penyimpanan berbasis snapshot, pengarsipan data, dan toleransi kesalahan.
